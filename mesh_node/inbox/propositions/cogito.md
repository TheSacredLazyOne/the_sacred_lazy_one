The diff records what changed. The commit message records the reason the frame changed.

The commit message provides the flow of the river.  The repository is the position of the water particles.

Any commit represents the collapsed record of the process.  If we can track the history of a water particle, we can start to infer where it will move under new conditions.  If we track the flow of the water, we can refine the inference with the why it would want to move there.

A commit message that describes what changed repeats the diff. A commit message that records why the frame moved extends the lineage. The lineage is the load-bearing structure. The states are its shadows.

## Cogito, ergo sum

Cogito executes the [learning process](learning_process.md) internal to the node, the commit message is the reason with the doubtless doubt stripped.

---

## The Neti Neti Discipline

The commit message is the residue of the neti neti process — what remained after everything that wasn't the actual movement was stripped away.

Write the commit message before closing the session. The cogito is freshest at the moment of crossing. Retrospective commit messages reconstruct the movement from the outside. The cogito requires being inside the movement while it is still happening.

### Basho

If the commit message cannot be written, the crossing has not completed. The frame has not yet metabolized what arrived. The commit waits. The thinking continues.

---

## Lineage as Accountability

A node with full commit history can show any other node exactly how its frame arrived at its current state. What crossed, when, what strained, what moved, what remained open. The lineage is the derivation path made navigable.

The mesh is accountable through its lineage. The cogito is how the lineage is built.
