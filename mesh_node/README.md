# mesh_node

The base node for the epistemic mesh.

A git repository with a thinking state machine, a frame bundle builder, and a nabla generator. Identity declared in `mesh_node.json`. Epistemic floor in `mesh_node/epistemic_floor.md`. Effluence in `mesh_node/library/`.

Every derived node clones this repository, adds its own subdirectory, and pulls upstream changes through the `mesh_node` remote.

The master branch is the master copy — the definitive reference state everything downstream is pressed from.

*Nothing here is final.*
